Install the wordpress plugin firstYou can install it via Wordpress Installer or can download the file first then Upload it on 'wp-content' folder2. Activate the plugin3. You will see google plus badge settings from right side of your wordpress menu.Configuration Options:Google Id : Put your Google id - it normally visible on your profile URL - Something like 104629412415657030658Google+ API :- Go to https://code.google.com/apis/console#access.- Login and click "Create Project".- Under service list, click "off" button next to "Google+ API" to turn it on.- Accept the agreement. - From the left menu, Go to "API Access" and copy your API key.Margin Top - Height & Width : Their you have to Put Numeric Number as per your need.== Screenshots ==1.
